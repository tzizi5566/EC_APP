## Highlights ##
* 项目进行模块化拆分，分为：核心组件、电商组件、UI组件。
* 单Activity设计。
* Retrofit2+RxJava2网络框架。
* Glide4图片加载库。
* GreenDao数据库。
* ZBar二维码扫描。
* JPush极光推送。
* ShareSDK多平台分享。
* 发现页面混合开发，WebView封装。
* MD风格商品详情页面。
